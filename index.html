ARIA uses your Google account to access Gemini AI for free. You need a one-time OAuth Client ID from Google Cloud — takes ~2 minutes.
▸ How to get your free Client ID (2 min)
  1. Open console.cloud.google.com and create or select a project
  2. Go to APIs & Services → Library, search Generative Language API and enable it
  3. Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID
  4. Application type: Web application
  5. Under Authorized JavaScript origins, add your site URL (e.g. https://psychologymatters561-design.github.io)
  6. Click Create, copy the Client ID, paste it below
